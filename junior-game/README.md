# 정우의 꼬마 거상 모험

초등학교 1학년도 할 수 있지만 너무 유아적이지 않은 독립형 경량 무역 어드벤처입니다. `starter/` full mode를 줄인 화면이 아니라, `junior-game/`에서 따로 실행되는 별도 게임입니다.

## 목표

- 튜토리얼로 첫 장사를 배운다.
- 전면 지도에서 도시를 직접 고른다.
- 지역 특산품을 사고판다.
- 이동 중 도적, 해적, 야생동물, 상인, 날씨 이벤트를 낮은 확률로 만난다.
- 이동 이벤트는 맞춤법 퀴즈로 지나간다.
- 돈을 모아 수레와 배를 장만한다.
- 300냥을 모으면 집으로 돌아가는 엔딩을 본다.

## 화면 구조

1. 도시/항구 화면: 큰 배경 일러스트와 4개 행동 버튼
2. 지도 화면: 이전 모드의 한국 전도형 래스터를 가져온 전면 지도
3. 장터 화면: 큰 상품 카드, 쉬운 가격, 짐 싣기 모션
4. 이동 화면: 수레/배가 약 5초 동안 지형을 따라 이동
5. 이벤트 화면: 큰 장면, 짧은 말풍선, 맞춤법 퀴즈
6. 탈것 장만 화면: 수레와 배 성장 카드
7. 엔딩 화면: 장부가 빛나고 집으로 돌아가는 선택

상단에는 정우, 도시, 돈, 별, 짐칸만 보입니다. 말풍선은 한 번에 1~2문장만 사용합니다.

## 핵심 루프

```text
도시 보기
→ 장터에서 물건 사기
→ 짐에 싣기
→ 지도에서 도시 고르기
→ 이동 중 사건 만나기
→ 처음 간 도시는 짧게 둘러보기
→ 장터에서 팔기
→ 돈 모아 수레/배 장만하기
→ 300냥 엔딩
```

## 지도 지역

지도에는 21개 지역이 들어 있습니다.

- 서울, 개성, 평양, 신의주
- 춘천, 강릉, 원산, 함흥, 청진
- 안동, 대구, 울산, 부산, 진주, 통영
- 전주, 광주, 순천, 여수, 목포
- 제주

현재 위치는 노란 테두리, 갈 수 있는 도시는 초록 강조, 선택 도시는 파동 링으로 구분합니다. 지도 배경은 라이트 전용 2D 전도인 `public/assets/maps/korea-light-map.svg`를 사용합니다.

지도는 바다, 육지, 산맥, 강, 제주도를 단순화해 보여줍니다. 도시를 누르면 지역명, 대표 물건 1~2개, 짧은 설명, 이동 수단 힌트가 아래 카드에 나옵니다.

## 상품과 특산품

상품은 8개만 사용합니다.

- 면포
- 건어물
- 소금
- 한지
- 귤
- 생선
- 약초
- 쌀

각 도시는 `buyGoodIds`와 `sellGoodIds`를 가집니다. 지역 특산품은 더 싸게 사고, 그 물건을 원하는 도시에서는 더 비싸게 팔 수 있습니다.

UI에는 `산 값`, `사는 값`, `파는 돈`, `여기서 사기 좋아`, `여기서 팔기 좋아` 같은 쉬운 표현만 씁니다. `매수가`, `매도가`, `마진`, `수익률`, `수요`, `공급`, `명성`, `신뢰도`는 쓰지 않습니다.

## 가격 변화

가격은 아이가 느낄 만큼만 조금 변합니다.

- 같은 도시에서 같은 물건을 여러 번 사면 사는 값이 조금 오른다.
- 같은 도시에서 같은 물건을 여러 번 팔면 파는 돈이 조금 내려간다.
- 특산품은 기본적으로 싸다.
- 먼 도시나 큰 장터에서는 더 잘 팔린다.

계산은 내부에서 처리하고, 화면에는 쉬운 말과 현재 값만 보여줍니다.

## 이벤트와 퀴즈

이동 한 번에 이벤트는 최대 1개만 나옵니다. 자유 플레이 이벤트는 종류별로 낮은 확률을 따로 굴립니다. 각 묶음은 3% 이하입니다.

- 나쁜 사건: 3%
- 좋은 사건: 2%
- 길 위 대화: 2%
- 이야기 단서: 1%

- 육로: 도적 맞춤법 퀴즈
- 바닷길: 해적 맞춤법 퀴즈
- 산길/들길: 야생동물 맞춤법 퀴즈
- 장터길: 상인 맞춤법 퀴즈
- 날씨: 비, 바람, 햇살 맞춤법 퀴즈
- 이야기: 장부가 빛나는 단서
- 설화: 떡 고개, 선녀의 옷감 같은 순한 이야기

맞춤법 퀴즈는 선택지 3개 중 바른 말을 고릅니다. 정답이면 별이나 돈을 받고, 오답이면 작은 돈 손해나 짐 손실만 있습니다.

`떡 하나 주면 안 잡아먹지`, `선녀와 나무꾼` 같은 소재는 무서운 장면이나 납치 표현을 쓰지 않고, `떡을 나누면 길이 환해진다`, `잃어버린 옷감을 찾아준다`처럼 초1용 서브스토리로 바꿔 사용합니다.

## 되팔기 규칙

짐 카드에는 처음 산 값을 함께 보여줍니다.

- 다른 도시에서 잘 팔리는 물건은 더 받을 수 있습니다.
- 같은 도시에서 바로 팔면 산 값보다 비싸게 팔리지 않습니다.
- 장터는 사고팔아도 자동으로 닫히지 않아 남은 돈과 짐칸을 바로 확인할 수 있습니다.
- 상품을 사면 아이콘이 짐칸으로 날아가고, 팔면 짐칸에서 빠지는 짧은 모션이 보입니다.

## 첫 방문 대화

처음 도착한 지역은 3~4컷으로 짧게 소개합니다.

- 지역 특징 1개
- 대표 특산품 1개
- 장사 힌트 1개

다시 방문할 때는 반복하지 않습니다.

## 저장

full mode와 분리된 저장 키를 씁니다.

```text
joseon_trade_junior_save_v1
```

저장 항목: 현재 도시, 돈, 별, 짐, 짐칸, 수레, 배, 열린 도시, 처음 방문한 도시, 튜토리얼 완료, 본 이벤트, 이야기 단서, 배지, 엔딩 여부, 지역별 가격 변화.

## 실행

```bash
cd junior-game
npm install
npm run build
npm run test
npm run dev -- --host 127.0.0.1 --port 4186
```

Playwright 테스트는 390x844, 412x915, 430x932 세로 화면과 844x390 가로 화면을 확인합니다.

## 전용 수레 에셋

`public/assets/vehicles/`에 junior 전용 투명 PNG 수레 4종을 둡니다.

- `junior-cart-bundle.png`: 보따리 수레
- `junior-cart-handcart.png`: 손수레
- `junior-cart-large.png`: 큰 수레
- `junior-cart-merchant.png`: 장사 수레

장터, 이동 화면, 탈것 장만 화면에서 같은 2D 동화풍 톤으로 사용합니다.
