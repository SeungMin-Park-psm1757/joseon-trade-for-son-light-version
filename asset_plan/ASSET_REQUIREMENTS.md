# 에셋 요구사항

## 1. MVP 에셋 원칙

- 첫 버전은 기능 우선이다.
- 외부 에셋이 없어도 CSS, SVG, 이모지, 단순 도형으로 플레이 가능해야 한다.
- 무료 2D 에셋은 나중에 적용한다.
- 라이선스가 불명확하면 사용하지 않는다.

## 3. 무료 에셋 우선 적용 방향

1. UI 아이콘과 임시 배 이미지는 Kenney 계열 CC0 에셋 우선 검토
2. 조선풍 지도/복식은 무료 에셋이 부족할 수 있으므로 처음에는 직접 SVG/단순 도형 사용
3. 향후 이미지 생성으로 조선풍 배·항구·지도 보강
4. 외부 에셋을 넣는 경우 반드시 `asset_plan/LICENSE_CHECKLIST.md`에 기록

## 4. 파일 구조 권장

```text
starter/public/assets/

```

각 에셋 폴더에는 출처와 라이선스를 적은 `README.md`를 둔다.
