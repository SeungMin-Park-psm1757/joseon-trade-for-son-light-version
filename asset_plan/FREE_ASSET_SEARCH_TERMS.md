## 4. 우선 후보

### Kenney

- Pirate Pack: 배/해적/해상 분위기 임시 활용
- UI Pack: 버튼/패널/아이콘 기본 UI
- Game Icons: 범용 아이콘
- Board Game Icons: 추상 지도 아이콘
- Tiny Town 또는 유사 타일: 도시/건물 임시 활용

### Game-icons.net

- SVG 아이콘 기반이므로 UI 배지에 적합
- 라이선스 확인 필요. 일반적으로 CC BY 계열이 많아 저작자 표시 필요 가능성이 있다.

### OpenGameArt

- 다양한 2D 에셋이 있지만 라이선스가 섞여 있다.
- GPL 계열은 웹앱 프로젝트에 불편할 수 있으므로 CC0 또는 CC BY 위주로 검토한다.

## 5. 피해야 할 것

- 라이선스가 없는 블로그 이미지
- 역사 지도 원본을 그대로 게임 배경으로 사용하는 것
- 상업 게임 스크린샷/아이콘 무단 사용
- AI 생성 이미지라도 특정 게임/작가 스타일을 직접 모방한 에셋
- 폰트 파일 무단 배포
