# 정우와 바람이 튜토리얼 설계

## 도입 이야기

주인공 정우는 현대의 초등학생이다. 어느 날 낡은 장부와 오래된 지도 조각을 보다가 갑자기 조선 후기 부산포 항구로 이동한다. 정우는 집으로 돌아갈 단서를 찾기 위해 조선의 장사길을 배우고, 작은 면포 장사부터 시작해 팔도상단의 장부 조각과 항구의 믿음을 모아 조선 최고의 거상으로 성장한다.

초반 튜토리얼은 긴 이야기보다 짧은 대화와 행동 유도를 우선한다.

1. 부산포에 도착했다는 사실을 이해한다.
2. 시장에서 면포를 살펴본다.
3. 면포를 산다.
4. 지도를 열고 대구로 간다.
5. 대구 시장에서 면포를 판다.
6. 돈을 모아 손수레를 목표로 삼는다.

## 요정 캐릭터

- 이름: 바람이
- 정체: 낡은 장부에 깃든 작은 바람 요정
- 역할: 정우에게 조선의 항구, 시장, 지도, 장비, 부탁을 알려주는 첫 친구
- 말투: “정우야”로 부르는 짧고 친절한 문장
- 문장 기준: 한 대화창 1~3문장, 한 문장 25자 안팎

## 튜토리얼 데이터

튜토리얼 대사는 `starter/public/data/tutorial_dialogues.json`에 둔다. 앱은 `GameData.tutorialDialogues`와 `tutorialDialogueById`로 읽는다.

핵심 필드:

- `id`: 대화 ID
- `stage`: 진행 단계
- `speaker`, `speakerName`: 말하는 사람
- `text`: 어린이용 짧은 대사
- `pauseGame`: 대화 중 게임 일시정지 여부
- `nextId`: 다음 대화
- `requiredAction`: 대화가 끝난 뒤 해야 할 행동
- `highlightTarget`: 강조할 버튼이나 상품 카드의 `data-testid`

## 진행 단계

| 단계 | 대표 대화 | 다음 행동 |
| --- | --- | --- |
| `intro` | “정우야, 눈을 떠봐!” | 이야기 진행 |
| `go_market` | “먼저 시장에서 면포를 보자.” | 시장 탭 |
| `inspect_good` | “상품 그림을 눌러봐.” | 면포 카드 |
| `buy_first_good` | “사기 버튼을 눌러보자.” | 면포 구매 |
| `open_map` | “더 비싸게 사줄 곳을 찾자.” | 지도 탭 |
| `choose_first_destination` | “대구 쪽에서 면포를 잘 사줘.” | 대구 이동 |
| `sell_first_good` | “대구 시장에서 면포를 팔아보자.” | 면포 판매 |
| `first_profit` | “첫 장사에 성공했어.” | 다음 목표 |
| `cart_goal` | “돈을 모으면 손수레를 살 수 있어.” | 장비 화면 |

## 게임 일시정지

`GameState`에 다음 필드를 추가했다.

- `tutorialStage`
- `completedTutorialSteps`
- `activeTutorialDialogId`
- `isPausedForDialog`
- `tutorialSkipped`

`activeTutorialDialogId`가 있고 `pauseGame`이 참이면 주요 입력을 막는다.

막는 행동:

- 시장 사기/팔기
- 지도 이동
- 장비 구매
- 배 수리/호출
- 동료 영입
- 부탁 납품
- 이벤트 선택
- 어업 시작

허용 행동:

- 대화의 다음 버튼
- 건너뛰기
- 다른 모달 닫기

발견 카드, 사건, 장비 알림처럼 더 높은 우선순위의 모달이 떠 있을 때는 해당 모달 조작을 막지 않는다.

## 어린이용 용어 기준

| 어려운 말 | 쉬운 표시 |
| --- | --- |
| 매수가 | 사는 값 |
| 매도가 | 파는 값 |
| 적재 | 실은 짐 |
| 적재칸 | 짐칸 |
| 예상 수익 | 벌 수 있는 돈 |
| 위험도 | 위험 |
| 항로 | 가는 길 |
| 교역 | 장사 |
| 의뢰 | 부탁 |
| 신뢰도 | 믿음 |
| 장부 | 장사 기록 |
| 도감 | 모은 기록 |

내부 데이터 ID와 테스트 ID는 유지하고, 화면 문구만 쉬운 말로 바꾸는 것을 원칙으로 한다.

## 검증 기준

- 새 게임 시작 후 발견 카드 다음에 정우 도입 대화가 나온다.
- 대화 중 주요 버튼은 실제 행동을 실행하지 않는다.
- 건너뛰기 후에는 튜토리얼 대화가 다시 나오지 않는다.
- 시장, 면포 구매, 지도, 대구 이동, 판매, 손수레 목표가 단계적으로 이어진다.
- `localStorage` 저장 후 이어하기에서도 튜토리얼 진행 상태가 유지된다.
