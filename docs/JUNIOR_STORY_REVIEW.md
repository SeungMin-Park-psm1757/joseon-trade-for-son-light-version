# Junior Story Safety Review

검수일: 2026-05-15

## 범위

- 메인 스토리 M01-M12
- 서브스토리 E01-E50
- 이야기 지도, 이야기 수첩, 대화, 선물 카드 문구
- 기존 junior 이동 퀴즈의 아이 노출 문구

## 1. 수정한 위험 표현

- 호랑이 메모의 원전 위험 표현을 배고픈 길동무와 떡을 찾는 친구로 바꿨다.
- 선녀 이야기 메모의 불편한 원전 설정은 지우고 `잃어버린 옷감 돌려주기`로만 남겼다.
- 심청전 메모의 슬픈 원전 표현은 `편지와 마음 전달`로 바꿨다.
- 임꺽정 메모는 거친 행동이 아니라 `정직함을 묻는 인물`로 정리했다.
- 홍길동, 충무공, 별주부전, 용왕 이야기의 거친 원전 메모를 공정함, 관찰, 편지, 화해 중심으로 낮췄다.
- 기존 이동 퀴즈의 거친 길막이 표시를 `길동무`, `말놀이`, `바다 말놀이꾼`으로 바꿨다.

## 2. 문장 길이 개선

- 서브스토리 대화 컷은 한 컷 1-2문장으로 유지했다.
- 자동 테스트에서 서브스토리 컷은 45자 이하, 메인 스토리 컷은 36자 이하로 제한한다.
- 어려운 지역 물건 라벨은 `이곳 물건`으로 낮췄다.
- 지도 상세의 선물 라벨은 `받은 것`, 선물 화면 제목은 `이야기 선물`로 바꿨다.

## 3. 이벤트 빈도 조정

- 튜토리얼 중에는 소문이 열리지 않는다.
- 소문 확률은 일반 진입 기준 30%를 유지한다.
- 이미 열린 소문이 있거나 이야기 대화 중이면 새 소문이 쌓이지 않는다.
- 체인 후속 이벤트는 바로 큰 대화로 튀어나오지 않고, 지도 소문으로만 열린다.
- 완료한 이벤트는 다시 시작되지 않는다.

## 4. 보상 밸런스 조정

- 일반 이벤트 별은 1개 중심으로 유지한다.
- 돈 선물은 8냥 이하로 제한한다.
- 장부 단서는 E25, E35, E42처럼 긴 체인 끝의 핵심 이벤트에만 둔다.
- 꾸미기는 이벤트 완료 뒤 별 상점에 열리는 방식으로 유지한다.
- 실패 보상은 큰 손실을 주지 않고 작은 돈/짐 손실 수준으로 제한한다.

## 5. 모바일 검수

- Playwright 프로젝트 기준 390x844, 412x915, 430x932, 844x390에서 테스트를 통과했다.
- 대화 컷, 선택지, 선물 카드, 이야기 수첩은 기존 테스트 화면 흐름 안에서 확인했다.
- 소문 지도 마커는 들은 이야기만 표시되어 과밀을 줄인다.
- 하단 탭은 늘리지 않고 도시 화면 보조 버튼으로 수첩을 연다.

## 6. 남은 한계

- 50개 이벤트 전체를 장시간 실제 플레이로 순차 확인한 것은 아니다.
- 지도 마커가 많이 열린 후의 밀도는 후속 시각 QA가 더 필요하다.
- 기존 `quiz_bandit`, `quiz_pirate` 타입 이름은 내부 호환을 위해 유지했다. 사용자 문구는 순화했다.
