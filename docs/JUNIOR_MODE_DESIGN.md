# Junior Mode Design

Last updated: 2026-05-14

## 목적

`정우의 꼬마 거상 모험`은 초등학교 1학년용 독립 게임이다. 기존 full mode의 쉬운 설정이 아니라 `junior-game/` 폴더에서 따로 실행되는 작은 무역 어드벤처다.

## 현재 방향

- 전면 화면형 도시/지도/장터/이벤트/수레와 배 장만 구조
- Galaxy S26 Ultra급 세로 모바일 웹 우선
- 라이트 전용 2D 한국 전도형 SVG 지도와 21개 지역
- 지역 특산품 기반의 간단한 사기/팔기
- 종류별 1~3% 저확률 이동 이벤트와 맞춤법 퀴즈
- 짐 싣기 모션과 약 5초 지형별 수레/배 이동
- 첫 방문 3~4컷 지역 소개
- 300냥 목표 엔딩

## Full Mode와 차이

| 항목 | Full Mode | Junior Game |
|---|---|---|
| 목표 | 깊은 교역 성장 RPG | 초1도 하는 작은 무역 어드벤처 |
| 실행 위치 | `starter/` | `junior-game/` |
| 화면 | 가로형 복합 UI | 세로형 전면 선택 화면 |
| 지도 | 정보가 많은 교역 지도 | 보드게임판 느낌의 한국 전도 |
| 장사 | 가격, 장부, 평판 포함 | 산 값/사는 값/파는 돈 중심 |
| 사건 | 복합 결과 가능 | 짧은 선택과 맞춤법 퀴즈 |
| 성장 | 여러 수치와 장비 | 수레와 배 중심 |
| 저장 | full mode 저장 | `joseon_trade_junior_save_v1` |

## 화면 구조

1. `JuniorIntroScreen`: 정우가 조선 장사 모험을 시작한다.
2. `JuniorCityScreen`: 도시/항구 배경이 화면 대부분을 차지하는 허브다.
3. `JuniorMapScreen`: 이전 모드의 전도형 지도 자산 위에서 도시를 선택하고 출발한다.
4. `JuniorTravelScreen`: 수레나 배가 경로를 따라 약 5초간 이동한다.
5. `JuniorVisitIntroScreen`: 첫 방문 지역을 3~4컷으로 소개한다.
6. `JuniorMarketScreen`: 큰 상품 카드로 사고팔며, 화면은 자동으로 닫히지 않는다.
7. `JuniorEventScreen`: 도적/해적/날씨/친절/이야기 이벤트를 전면 화면으로 처리한다.
8. `JuniorShopScreen`: 수레와 배를 카드로 장만한다.
9. `JuniorEndingScreen`: 300냥 목표 달성 후 집으로 돌아간다.

## 2026-05-14 수레와 배/지역 학습 보강

- 수레와 배 화면은 `현재 수레와 배 → 다음 목표 → 수레 장만 → 배 장만` 순서로 읽힌다.
- 현재 보유 수레와 배를 따로 보여준다: `수레: 큰 수레 · 땅길 짐칸 4칸`, `배: 작은 나룻배 · 바닷길 짐칸 2칸`.
- 모든 수레/배 카드에는 가격, 짐칸, 쉬운 설명, 상태 뱃지를 표시한다.
- 상태 뱃지는 `쓰는 중`, `보유 중`, `살 수 있어`, `돈이 부족해`를 사용한다.
- 지도 길 카드에는 육로/해로에 따라 어떤 짐칸을 쓰는지 표시한다.
- 배가 없는데 해로를 고르면 출발은 막고, `작은 나룻배를 장만하면 갈 수 있어`를 보여준다.
- 하단 상태 영역에는 현재 땅길/바닷길 짐칸 요약을 항상 표시한다.
- 도시/장터 진입 흐름에는 지역 학습 이벤트가 붙는다. 튜토리얼 중에는 나오지 않는다.
- 지역 이벤트 유형은 상인 소문, 사투리 한마디, 명소 이야기, 계절 이야기다.
- 사투리는 짧게 쓰고, 바람이가 뜻이나 느낌을 표준어로 풀어준다.

## 지도 지역

지도는 실제 지리 정확도보다 “한국을 돌아다닌다”는 감각을 우선한다. 북부 지역도 포함해 지리 학습 효과를 높였다.

- 서울, 개성, 평양, 신의주
- 춘천, 강릉, 원산, 함흥, 청진
- 안동, 대구, 울산, 부산, 진주, 통영
- 전주, 광주, 순천, 여수, 목포
- 제주

현재 위치는 노란 테두리, 갈 수 있는 도시는 초록 강조, 선택 도시는 파동 링과 scale feedback으로 구분한다. 바닷길은 다른 선 스타일로 표현한다. 지도 배경은 `junior-game/public/assets/maps/korea-light-map.svg`를 사용한다.

지도는 hand-painted casual map 톤을 목표로 한다. 한국 전도형 실루엣, 바다/육지 구분, 제주도, 산맥, 압록강/두만강 힌트, 동해/서해/남해 라벨을 단순화해 보여준다.

도시 선택 카드에는 지역명, 도시명, 대표 물건 1~2개, 한 줄 지역 설명, 수레/배 이동 힌트를 표시한다.

## 지역별 특산품

상품은 8개다.

- 면포
- 건어물
- 소금
- 한지
- 귤
- 생선
- 약초
- 쌀

각 도시는 `buyGoodIds`와 `sellGoodIds`를 가진다. UI에는 `여기서 사기 좋아`, `서울에서 인기 많아`, `대구 장터에서 잘 팔려`, `산 값`, `파는 돈`처럼 자연어 중심 표현만 보여준다.

## 매매 규칙

- 시작 돈은 30냥, 시작 짐칸은 2칸이다.
- 상품 1개는 짐칸 1칸이다.
- 상품을 사면 아이콘이 카드에서 짐칸으로 이동하고 짐칸이 튄다.
- 장터는 구매/판매 뒤 자동으로 닫히지 않는다.
- 같은 도시에서 같은 물건을 많이 사면 사는 값이 조금 오른다.
- 같은 도시에서 같은 물건을 많이 팔면 파는 돈이 조금 내려간다.
- 특산품은 더 싸고, 원하는 도시에서는 더 비싸게 팔린다.
- 짐 카드에는 처음 산 값을 보여준다.
- 같은 도시에서 바로 되팔 때는 산 값보다 비싸게 팔리지 않는다.

## 이벤트와 맞춤법 퀴즈

`JUNIOR_EVENTS`는 다음 구조를 수용한다.

- `type`: travel, weather, kindness, story, growth, ending, quiz_bandit, quiz_pirate
- `fairyText`: 요정 말풍선
- `choices`: 1~3개 선택지
- `quiz`: 문제, 선택지, 정답, 정답/오답 결과
- `reward`: 별, 돈, 이야기 단서, 짐 손실

자유 플레이 이벤트는 총합 3%가 아니라 종류별 저확률 묶음으로 굴린다. 각 묶음은 3% 이하다.

- 나쁜 사건: 3%
- 좋은 사건: 2%
- 길 위 대화: 2%
- 이야기 단서: 1%

육로 이동은 도적/야생동물/상인/날씨 맞춤법 퀴즈, 해상 이동은 해적/상인/날씨 맞춤법 퀴즈를 우선 발생시킨다. 정답이면 무사 통과, 오답이어도 게임오버 없이 작은 손해만 준다. 이벤트 풀은 20개 이상의 맞춤법 퀴즈를 가진다.

설화 소재는 공포나 납치 장면으로 쓰지 않는다. `떡 하나 주면 안 잡아먹지`는 `떡을 나누면 고개 길이 환해지는 이야기`, `선녀와 나무꾼`은 `선녀의 잃어버린 옷감을 찾아주는 이야기`로 바꿔 서브스토리화한다.

## 튜토리얼

튜토리얼은 새 화면 구조에 맞춰 실습형으로 구성한다.

1. 도시에서 장터 가기
2. 물건 하나 사기
3. 짐에 싣기 모션 보기
4. 지도에서 대구 선택
5. 이동 중 맞춤법 퀴즈
6. 첫 방문 지역 소개
7. 장터에서 팔기
8. 돈/별과 다음 성장 목표 확인

대사는 짧은 요정 말투로 유지하고, 복합 UI 설명문은 쓰지 않는다.

## 성장 구조

- 보따리 수레: 시작, 짐칸 2칸
- 손수레: 100냥, 짐칸 3칸
- 큰 수레: 190냥, 짐칸 4칸
- 장사 수레: 300냥, 짐칸 5칸
- 작은 나룻배: 200냥, 바닷길 도움
- 작은 돛배: 360냥, 먼 바닷길에 든든함

UI에는 `짐을 하나 더 실어`, `바닷길을 건널 때 좋아`, `먼 길도 든든해`처럼 자연스러운 짧은 문장을 쓴다.

## 숨긴 시스템

명성, 신뢰도, 장부 목록, 도감, 위험도, 평균 매입가, 수익률, 수요/공급, 세금, 허가장, 내구도, 수리비, 복잡한 장비 수치표는 junior UI에 표시하지 않는다.
