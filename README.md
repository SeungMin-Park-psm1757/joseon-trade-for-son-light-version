# 정우의 꼬마 거상 모험

초등학교 1학년도 할 수 있는 모바일 세로형 경량 무역 어드벤처입니다. 기존 full mode와 섞이지 않는 라이트버전 전용 저장소입니다.

## 실행 링크

GitHub Pages 배포 링크:

[https://seungmin-park-psm1757.github.io/joseon-trade-for-son-light-version/](https://seungmin-park-psm1757.github.io/joseon-trade-for-son-light-version/)

로컬 실행:

[http://127.0.0.1:4390/](http://127.0.0.1:4390/)

```bash
npm install
npm run dev
```

## 게임 목표

- 정우가 조선에서 바람이와 함께 첫 장사를 배웁니다.
- 한국 전도 느낌의 지도에서 도시를 고르고 이동합니다.
- 도시별 특산품을 사고팔며 돈을 모읍니다.
- 이동 중 도적, 해적, 날씨, 상인, 동화 이벤트를 만납니다.
- 맞춤법 퀴즈로 사건을 지나갑니다.
- 돈을 모아 수레와 배를 장만합니다.
- 300냥을 모으면 집으로 돌아가는 엔딩을 볼 수 있습니다.

## 화면 구조

1. 시작 / 이어하기
2. 도시 화면
3. 지도 화면
4. 장터 화면
5. 이동 화면
6. 이벤트 / 맞춤법 퀴즈 화면
7. 탈것 장만 화면
8. 엔딩 화면

모바일 세로 화면을 우선합니다. 주요 검수 화면은 `360x800`, `390x844`, `412x915`, `430x932`입니다.

## 장사 루프

```text
도시 보기
-> 장터에서 물건 사기
-> 짐에 싣기
-> 지도에서 도시 고르기
-> 이동 중 사건 만나기
-> 첫 방문 도시 소개 보기
-> 장터에서 팔기
-> 돈을 모아 수레와 배 장만하기
-> 300냥 엔딩
```

## 쉬운 표현 원칙

사용하는 말:

- 산 값
- 파는 돈
- 여기서 사기 좋아
- 여기서 팔기 좋아
- 짐
- 돈
- 별
- 탈것 장만

사용하지 않는 말:

- 매수가
- 매도가
- 마진
- 수익률
- 수요 / 공급
- 명성
- 신뢰도
- 장부 UI

## 아트 방향

- 밝은 2D 동화풍 무역 모험 게임 톤
- 21개 도시별 고유 배경
- 통일된 상품 아이콘 8종
- 수레 / 배 성장 에셋
- 주요 이벤트 컷인 10종
