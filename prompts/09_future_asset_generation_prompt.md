# 후속 이미지 생성/에셋 보강 프롬프트

이 프롬프트는 1차 교역 루프가 작동한 뒤, 무료 에셋이 부족한 부분을 조선풍 2D 이미지로 보강할 때 사용합니다.

## 사용 전 조건

- 교역/이동/저장/이벤트 기본 루프가 작동해야 합니다.
- 무료 에셋 후보와 라이선스를 먼저 확인해야 합니다.
- 이미지 생성 결과를 사용할 경우 스타일 프롬프트를 `.prompts/`에 저장해야 합니다.

## 스타일 방향

```text
조선 후기 무역 RPG용 2D 모바일 웹게임 에셋.
따뜻한 한지색 배경, 간결한 보드게임풍 아이콘, 낮은 디테일, 작은 화면에서 잘 읽히는 실루엣.
역사 고증은 분위기 중심, 과도한 사실적 묘사보다 명확한 UI 가독성 우선.
```

## 에셋 배치 후보

- 작은 돛단배
- 연안 상선
- 손수레
- 소달구지
- 항구 아이콘
- 내륙 도시 아이콘
- 갯벌 아이콘
- 암초 아이콘
- 태풍 아이콘
- 도적/해적 이벤트 아이콘

## Codex 요청 문장

```text
위 스타일을 유지해서 필요한 에셋 프롬프트 묶음을 작성해 주세요. 실제 생성 전에 각 프롬프트를 `.prompts/asset_prompts.md`에 저장하고, 생성된 에셋은 `starter/public/assets/generated/`에 배치하되 라이선스/출처 메모를 함께 남기세요.
```
