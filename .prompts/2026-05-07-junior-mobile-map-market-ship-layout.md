# Junior light version mobile UX correction prompt

## 목적

정우의 꼬마 거상 모험 라이트버전은 초등 저학년이 휴대폰 세로 화면에서 바로 이해하고 조작할 수 있는 작은 무역 모험 게임이어야 한다. 이번 수정은 기능 추가가 아니라 실제 휴대폰 화면에서 발생한 구조 문제를 바로잡는 작업이다. 기준 화면은 사용자가 제공한 모바일 브라우저 캡처이며, 주소창과 하단 브라우저 UI가 보이는 실제 사용 환경을 전제로 한다.

이번 작업에서 실패로 보는 기준은 명확하다. 글자가 잘리거나, 카드가 화면 밖으로 밀리거나, 첫 화면이 한쪽으로 쏠리거나, 지도 설명 패널이 지도를 밀어내거나, 요정 대화가 화면 맨 아래에서 뒤늦게 보이거나, 배 그림이 단계별 차이를 주지 못하면 실패다. “작동한다”는 이유로 통과하지 않는다. 아이가 실제로 봤을 때 지금 무엇을 눌러야 하는지 알고, 게임 화면이 완성된 것처럼 보여야 한다.

## 모바일 레이아웃 원칙

1. 상단 정우 상태바 바로 아래에 요정/미션 메시지를 둔다. 요정은 하단 보조 장식이 아니라 현재 행동을 알려주는 안내자다. 따라서 대화/미션은 항상 상단 상태바 다음에 나오고, 도시/장터/지도/탈것 화면의 본문은 그 아래에 배치한다.
2. 하단에는 빠른 이동 네비게이션과 목표 바만 둔다. 이 둘이 본문 카드와 겹치면 안 된다.
3. 본문 영역은 잘림보다 스크롤을 우선한다. 장터와 탈것 화면은 카드 수가 많기 때문에 본문 내부 스크롤을 허용하되, 버튼과 가격 정보가 카드 하단에서 잘리지 않도록 카드 높이를 낮추고 정보 밀도를 다시 조정한다.
4. 주요 터치 버튼은 56px 이상을 유지한다. 단, 카드 내부 보조 버튼/가격 배지는 큰 버튼처럼 보여도 텍스트가 잘리지 않게 줄여야 한다.
5. 첫 화면은 오른쪽 쏠림을 제거한다. 정우 일러스트가 화면 오른쪽을 과도하게 차지하면 제목과 시작 버튼의 중심이 흔들린다. 배경과 정우는 2D 게임 키비주얼처럼 보이되, 정우의 얼굴과 몸이 화면 중앙-하단에 안정적으로 들어오게 한다.

## 지도 화면 원칙

지도 화면은 지도 자체가 주인공이어야 한다. 별도 큰 설명 패널이 지도 아래를 차지해서는 안 된다. 도시를 누르면 해당 도시 옆에 작은 말풍선형 정보 카드가 뜬다. 말풍선에는 도시명, 도시 타입, 대표 물건 1~2개, 이동 가능 여부, 출발 버튼만 들어간다. 너무 긴 설명은 지도 화면에서 제거한다.

도시 간 연결은 현재처럼 일부 루트만 보이는 보드게임식 제한에서 벗어나야 한다. 태백산맥처럼 지형 장벽이 있는 일부 동서 연결은 신중하게 제한하되, 대부분의 인접 도시끼리는 이동할 수 있어야 한다. 인접 도시와 해안 항구 간 해로는 충분히 열어 둔다. 아이가 “왜 바로 옆 도시인데 못 가지?”라고 느끼면 실패다.

육로는 수도권, 서북, 관서, 관북, 관동, 영남, 호남, 남해안을 자연스럽게 잇는다. 해로는 서해 항구, 남해 항구, 동해 항구, 섬 거점을 연결한다. 배가 없으면 해로는 잠기지만, 지도에서는 왜 배가 필요한지 알 수 있게 표시한다.

## 장터 화면 원칙

장터 카드에서 글자가 잘리면 안 된다. 상품 카드는 2열을 유지하되, 카드 안 정보는 상품 그림, 이름, 사기/팔기 좋음, 사는 값/파는 값, 내 짐 수량 정도로 제한한다. 카드 하단 가격 배지가 화면 아래로 잘리면 카드 구조를 다시 줄인다. 카드 내부의 큰 아이콘은 유지하되, 아이콘이 텍스트를 밀어내지 않도록 한다.

“여기서 사기 좋아” 같은 문구는 유지하되, 너무 길어지는 문구는 “사기 좋아”, “값이 좋아”, “내 짐 1개”처럼 짧게 줄인다. 초등 저학년에게 쉬운 말이어야 하지만 유아용 말투가 되면 안 된다.

## 탈것/배 화면 원칙

배는 수레처럼 단계별 차이가 시각적으로 보여야 한다. 작은 나룻배, 작은 돛배, 튼튼한 돛배, 장사배는 같은 그림에 깃발만 바꾼 수준이면 실패다. 이번 수정에서는 최소한 UI 카드 안에서 배 단계가 명확히 다른 2D 도식으로 보이게 한다. 작은 나룻배는 낮고 단순한 배, 작은 돛배는 돛이 보이는 배, 튼튼한 돛배는 보강된 선체와 큰 돛, 장사배는 짐칸과 장사 깃발이 분명하게 보여야 한다.

이미지 자체 교체가 충분하지 않다면 카드 내부에 2D 도식 레이어를 추가해 단계 차이를 보강한다. 단, 그림판 낙서풍이나 임시 도형처럼 보이면 안 된다. 선체, 돛, 짐칸, 깃발, 물결 레이어가 하나의 게임 카드 아트처럼 보이게 구성한다.

## 완료 검수

- 첫 화면 정우 일러스트가 오른쪽으로 과하게 쏠리지 않는다.
- 요정/미션 메시지가 상단 상태바 바로 아래에 온다.
- 장터 카드의 가격, 상품명, 버튼 글자가 모바일에서 잘리지 않는다.
- 탈것 카드가 모바일에서 위아래로 잘리지 않고 스크롤 가능하다.
- 지도 화면은 지도 중심이고, 도시 설명은 선택 도시 옆 말풍선으로 뜬다.
- 인접 도시 이동 루트가 충분히 열려 있다.
- 배 단계 차이가 한눈에 보인다.
- 390x844, 412x915, 430x932에서 직접 캡처하여 확인한다.
