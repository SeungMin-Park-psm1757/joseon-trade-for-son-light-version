# 2026-05-05 Junior Map Trade Events Ending Pass

## 1. 변경 요약

- `정우의 첫 장사놀이`를 `정우의 꼬마 거상 모험`으로 확장했다.
- 첫 장사 튜토리얼 뒤 자유 플레이가 열리도록 바꿨다.
- 지도, 도시 이동, 지역 특산품, 매매, 이동 이벤트, 숫자퀴즈, 수레/배 성장, 엔딩을 `junior-game` 안에만 구현했다.
- full mode 컴포넌트, 저장 키, CSS를 import하지 않았다.

## 2. 지도 이동 구현

- `KoreaMiniMap`으로 세로형 대한민국 전도 느낌의 보드게임판 지도를 만들었다.
- 도시 버튼은 부산, 대구, 전주, 목포, 통영, 서울, 제주, 강릉을 표시한다.
- 현재 도시는 노란색으로 반짝이고, 갈 수 있는 도시는 초록색이다.
- 제주 같은 어려운 도시는 배 구매 전에는 잠금처럼 보인다.
- 수레 길은 갈색 선, 배 길은 파란 점선이다.

## 3. 지역 특산품 구현

- 상품 8개를 구현했다: 면포, 건어물, 소금, 한지, 귤, 생선, 약초, 쌀.
- 도시별 `buyGoodIds`, `sellGoodIds`로 사기 좋은 물건과 팔기 좋은 물건을 분리했다.
- UI 문구는 `사기 좋아`, `팔기 좋아`, `사는 돈`처럼 쉬운 말만 사용한다.

## 4. 매매 시스템 구현

- 시작 돈은 30냥이다.
- 시작 짐칸은 2칸이다.
- 상품 1개는 짐칸 1칸을 사용한다.
- 돈 부족과 짐칸 부족 메시지를 짧게 보여준다.
- 팔기 좋은 도시에서 팔면 돈과 별이 늘어난다.

## 5. 이벤트/숫자퀴즈 구현

- `JUNIOR_EVENTS`에 50개 이벤트를 준비했다.
- 도적 숫자퀴즈 5개와 해적 숫자퀴즈 5개를 실제 이동 이벤트로 사용한다.
- 퀴즈는 1~20 범위의 덧셈/뺄셈과 3개 선택지로 구성했다.
- 정답은 별 또는 돈을 주고, 오답도 게임오버 없이 작은 손해만 준다.

## 6. 수레/배 성장 구현

- 보따리, 손수레, 큰 수레를 구현했다.
- 작은 나룻배, 작은 돛단배를 구현했다.
- 손수레는 80냥, 큰 수레는 150냥, 작은 나룻배는 120냥, 작은 돛단배는 220냥이다.
- 수레는 짐칸을 늘리고, 배는 제주 이동을 열어준다.

## 7. 엔딩 구현

- 300냥을 모으면 `집으로` 버튼이 열린다.
- 장부가 문처럼 열리는 엔딩 선택 화면을 보여준다.
- `집으로 가기`를 누르면 `꼬마 거상 정우` 엔딩 화면으로 간다.
- 엔딩 후 다시 하기, 계속 장사, 큰 모험 보기를 제공한다.

## 8. 테스트 결과

- `cd junior-game && npm run build`: 통과.
- `cd junior-game && npm run test`: 28개 통과.
- `cd starter && npm run build`: 통과.
- `JUNIOR_EVENTS` 이벤트 수: 50개 확인.
- `starter/src`에서 junior import, `gameMode`, `joseon_trade_junior_save_v1` 참조 없음 확인.
- 후속 점검에서 안동/여수 특산품을 지시 표에 맞게 보정하고, 10개 도시 버튼을 지도에 표시했다.
- 가로 보조 화면에서 잠긴 도시 버튼이 클릭을 가로막지 않도록 비활성 버튼 포인터 처리를 보정했다.

## 10. 남은 한계

- 지도는 CSS/SVG 기반 보드게임판이므로 실제 지리 정밀 지도는 아니다.
- 이벤트는 약 40% 느낌의 가벼운 조건으로 발생하고, 발생 시 아직 안 본 이벤트를 우선 보여준다. 추후 더 자연스러운 랜덤 덱으로 조정할 수 있다.
- 강릉, 안동, 여수는 데이터와 일부 연결을 준비했지만 화면에는 8개 도시만 우선 표시한다.
- 일부 이벤트 장면은 텍스트 아이콘 스타일이며, 별도 원화 컷신은 아직 없다.
