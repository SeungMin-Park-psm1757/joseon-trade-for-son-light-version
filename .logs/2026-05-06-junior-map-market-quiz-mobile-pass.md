# 2026-05-06 Junior Map/Market/Quiz/Mobile Pass

## 1. 변경 요약

- `junior-game/` 독립 게임을 전면 화면형 작은 무역 어드벤처로 보강했다.
- 도시/항구, 지도, 장터, 이동, 이벤트, 첫 방문 소개, 탈것 장만, 엔딩 화면을 분리했다.
- 한국 전도 느낌의 지도 지역을 21개로 확장했다.
- 숫자퀴즈 대신 도적/해적 맞춤법 퀴즈를 적용했다.
- 장터 구매/판매 후 화면이 자동으로 닫히지 않도록 바꿨다.
- 지역별 가격 변화와 짐 싣기 모션을 추가했다.

## 2. 지도 이동 구현

- 지도는 `JuniorMapScreen`의 전면 보드게임판으로 구현했다.
- 현재 위치, 갈 수 있는 도시, 선택 도시를 색상과 테두리, 파동 링으로 구분한다.
- 부산에서는 기본 추천 목적지를 대구로 두어 통영/대구 버튼 겹침을 줄였다.
- 수레/배 이동은 `JuniorTravelScreen`에서 약 1.15초 동안 ease-in-out으로 진행된다.

## 3. 지역 특산품 구현

- 지역: 서울, 개성, 평양, 신의주, 춘천, 강릉, 원산, 함흥, 청진, 안동, 대구, 울산, 부산, 진주, 통영, 전주, 광주, 순천, 여수, 목포, 제주.
- 상품: 면포, 건어물, 소금, 한지, 귤, 생선, 약초, 쌀.
- 각 도시는 `buyGoodIds`, `sellGoodIds`, `introLines`를 가진다.

## 4. 매매 시스템 구현

- `getBuyPrice`, `getSellPrice`로 지역 특산품과 판매 선호 도시를 반영한다.
- 같은 도시에서 같은 물건을 반복 구매하면 사는 값이 조금 오른다.
- 같은 도시에서 같은 물건을 반복 판매하면 파는 값이 조금 내려간다.
- UI에는 `사는 값`, `파는 값`, `여기서는 싸요`, `비싸게 팔려요`만 표시한다.

## 5. 이벤트/맞춤법 퀴즈 구현

- 육로 이동은 도적 맞춤법 퀴즈를 우선 발생시킨다.
- 해상 이동은 해적 맞춤법 퀴즈를 우선 발생시킨다.
- 정답은 별/돈 보상, 오답은 작은 돈 손해나 짐 손실로 처리한다.
- 실패해도 게임오버는 없다.

## 6. 튜토리얼 수정

- 장터 열기, 물건 사기, 짐 싣기, 지도 이동, 이동 이벤트, 첫 방문 소개, 팔기까지 실습형 흐름으로 이어진다.
- 튜토리얼 대사는 요정이 직접 말하는 짧은 문장으로 유지했다.

## 7. 모바일 UX 검수

- 검수 viewport: 390x844, 412x915, 430x932, 844x390.
- 모든 검수 화면에서 `scrollWidth == viewport width`, `scrollHeight == viewport height`.
- 48px 미만 버튼 없음.
- 390x844 지도/장터/이벤트 화면 잘림 없음.
- Galaxy S26 Ultra급 430x932 지도 화면 잘림 없음.
