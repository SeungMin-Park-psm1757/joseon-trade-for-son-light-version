# 2026-04-28 Full Polish Roadmap Pass

## 1. 변경 요약

- 18개 전 항구의 `port_flavors.json` 데이터를 작성했다.
- 시장 거래를 더 큰 아이콘, 큰 사기/팔기 버튼, 명확한 플로팅 피드백 중심으로 다듬었다.
- 첫 30분 이후 제주/대마도까지 이어지는 장기 성장 지도 노드를 추가했다.
- 사건 결과 칩에 CSS 텍스트 대신 작은 픽셀 PNG 아이콘을 연결했다.
- 장비 구매/배 호출 축하 모달, 배 정박 위치 표시, 저장 슬롯 3개, 도감 요약, 계절 소식 그림, 시장 BGM 패턴을 추가했다.

## 2. 항구별 개성

`starter/public/data/port_flavors.json`은 이제 `ports.json`의 18개 항구 전체를 가진다.

- 항구별 간판 제목/설명
- 시설별 NPC 대사
- 지역 소문
- 대표 상품
- 시장 진열 우선순위
- 상품별 장터 좌표

가격/수요 계산은 기존 로직을 유지했고, 데이터는 화면 표현에만 쓰인다.

## 3. 시장 UI

- 상품 카드의 특산/수요/일반과 사기/팔기 추천 배지를 분리했다.
- 거래 팝업에서 수량 설명을 줄이고 사기/팔기 큰 버튼 2개만 남겼다.
- 거래 피드백은 `+면포 -25냥`, `-면포 +25냥` 형태로 표시한다.
- 터치 대상과 상품 아이콘 크기를 키웠다.

## 4. 성장 지도와 장비 연출

의뢰 탭 성장 지도는 다음 흐름까지 확장했다.

면포 구매 → 대구 이동 → 면포 판매 → 소금 장사 → 손수레 → 어업 → 어선 → 남해 거점 → 제주 준비 → 제주 도착 → 연안 상선 → 왜관 허가 → 대마도

장비 구매와 배 호출 완료 시 작은 축하 모달이 뜨고 다음 행동으로 이어진다.

## 5. 이벤트/배 호출/사운드

- `starter/public/assets/result-icons/`에 결과 칩용 PNG 9개를 생성했다.
- 사건 결과는 돈/배/수레/화물/정박 안전을 아이콘으로 구분한다.
- 전체 지도에서 배 정박 항구에 작은 배 표시를 추가했다.
- 시장 전용 BGM 패턴을 추가해 항구/시장/지도 분위기를 나눴다.

## 6. 도감/저장/계절

- 화물/장부 화면에 방문 항구, 거래 상품, 만난 NPC 수를 보여주는 상단 도감을 추가했다.
- 시작 화면에 1~3번 저장 슬롯을 추가했다. 1번 슬롯은 기존 저장 키를 유지한다.
- 새 게임 시작 시 기존 슬롯에 저장이 있으면 확인한다.
- 월간 소식 카드에 봄/여름/가을/겨울 픽셀풍 작은 그림을 붙였다.

## 9. 남은 한계

- 저장 슬롯은 로컬 브라우저 슬롯 3개이며, 이름 변경이나 썸네일은 없다.
- 장기 성장 지도는 노드형 안내이고, 별도 챕터 컷신은 아니다.
- 이벤트 장면 자체는 CSS 픽셀풍이고, 결과 칩 아이콘만 PNG다.
- 전 항구 시장 좌표는 들어갔지만, 항구별 전용 배경 레이어까지 세분화되지는 않았다.
