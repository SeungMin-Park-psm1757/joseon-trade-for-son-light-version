# 2026-05-07 Light Phase 4 Travel Event

## 1. 변경 요약

- 이동 화면을 단순 도형/빠른 이동에서 출발 카드, 경로 안내, 진행 바, 도착 예고가 있는 전면 이동 장면으로 재구성했다.
- 수레/배 이동 시간을 약 5초로 늘리고, 이동 중간 프레임이 눈에 보이도록 했다.
- 들길/산길/강가길/해안길/바닷길/제주 뱃길/북방길을 구분할 수 있는 CSS 2D 일러스트 레이어를 추가했다.
- 이벤트 화면에 요정 대사를 별도 대화 카드로 표시했다.
- 첫 방문 지역 소개에 정우와 요정 초상, 화자 표시, 컷 진행을 추가했다.

## 2. 이동 씬 구현

- `JuniorTravelScreen`에 출발 카드, 경로 설명 카드, 목적지 카드, 이동 힌트 말풍선, 진행 바를 추가했다.
- 수레는 흔들림과 그림자를 유지하며 천천히 이동한다.
- 배는 파도 위에서 흔들리는 bob 모션을 쓴다.
- 바닷길에는 섬과 물결 레이어, 강가길에는 다리/갈대, 산길에는 능선/고개 레이어, 북방길에는 차가운 산/강 색감을 적용했다.

## 3. 이벤트/퀴즈 연결

- 기존 이벤트 발생 흐름은 유지했다.
- 이동 후 이벤트가 있으면 일반 조작이 멈추고 전면 이벤트 화면으로 들어간다.
- 이벤트 화면은 큰 배경, 정우/요정, 사건 아이콘, 요정 대사, 맞춤법 퀴즈 선택지로 구성된다.
- 도적/해적/동물/상인/설화 이벤트는 기존 1~3% 저확률 묶음 정책을 유지한다.

## 4. 첫 방문 대화

- 첫 방문 지역은 3~4컷 소개 대화를 보여준다.
- 화자는 `바람이`와 `정우`로 표시된다.
- 대화 중 일반 조작은 막히며 `다음` 버튼으로만 진행한다.
- 지역 특산품, 분위기, 다음 행동 힌트가 짧게 나온다.

## 5. 요정 메시지

- 이동 중 하단 요정 말풍선이 목적지와 경로 힌트를 말하도록 바뀌었다.
- 이벤트 화면에도 `바람이` 대사를 별도 말풍선으로 보여준다.
- 실패/성공 흐름은 기존처럼 `eventResult`에서 짧은 결과와 확인 버튼으로 처리한다.

## 8. 남은 한계

- 이동 배경은 CSS 기반 2D 일러스트 레이어다. 도시별 완전 고유 이동 컷신까지는 아직 아니다.
- 수레 바퀴 회전은 이미지 자체가 단일 PNG라 별도 분리 애니메이션이 없다.
- 이후 배경 전용 PNG/WEBP를 제작하면 더 고급스러운 이동 씬으로 끌어올릴 수 있다.
