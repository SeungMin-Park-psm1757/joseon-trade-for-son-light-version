# 2026-04-28 Port Growth Event Pass

## 1. 변경 요약

- 부산포, 대구, 전주, 목포, 제주, 대마도에 고유 항구 생활감 문구와 대표 상품을 추가했다.
- 항구 화면 우측에 `오늘의 항구` 간판을 추가했다.
- 시설 NPC 대사가 현재 항구 성격에 맞게 바뀌도록 했다.
- 첫 길잡이를 “첫 30분 성장길”로 확장했다.
- 이벤트 선택지를 그림 아이콘이 있는 버튼으로 개선했다.

## 2. 항구별 고유 생활감

`App.tsx`의 `PORT_FLAVORS` 매핑으로 초반 핵심 항구의 특징을 관리한다.

- 부산포: 왜관 길목, 면포/건어물/인삼
- 대구: 영남 내륙 장터, 소금/건어물/미역
- 전주: 한지와 곡식, 쌀/한지/부채
- 목포: 서남해 갯벌, 소금/홍어/건어물
- 제주: 바람 센 섬, 말/감귤/미역
- 대마도: 허가가 필요한 바깥섬, 은/일본 종이/칠기

## 3. 첫 30분 성장선

기존 첫 10분 튜토리얼을 `첫 30분 성장길`로 확장했다.

순서:

1. 면포 2개 구매
2. 대구 이동
3. 면포 판매
4. 소금 장사
5. 손수레
6. 어업
7. 어선
8. 제주 준비

각 단계는 상품/지도/장비 아이콘을 사용하며, 완료/현재/잠금 상태를 유지한다.

## 4. 이벤트 선택지 그림 버튼

`choiceVisualKind`와 `choiceHint`를 추가해 기존 이벤트 선택지를 그림 버튼으로 표시한다.

지원 표현:

- 전투
- 우회/도주
- 수리
- 돈/세금
- 짐 손실
- 휴식/대기
- 허가

이벤트 로직은 바꾸지 않고 표시만 강화했다.

## 7. 남은 한계

- 항구별 고유 문구는 코드 매핑이며, 장기적으로는 데이터 JSON으로 옮길 수 있다.
- 첫 30분 길잡이는 844×390에 맞춰 매우 compact하게 구성되어 있어, 더 깊은 설명은 별도 의뢰 화면으로 분리하는 편이 좋다.
- 이벤트 선택지 아이콘은 CSS 픽셀풍이며, 전용 이벤트 일러스트는 아직 없다.
