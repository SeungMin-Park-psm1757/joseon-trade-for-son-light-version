# 2026-04-29 Scene-First UI Pass 1

## 1. 변경 요약

- 항구 메인, 지도/이동, 시장 화면을 장면 중심 UI로 재정비했다.
- 한반도 전도 기반 SVG 지도 레이어를 추가했다.
- 항구 시설을 배경 위 hotspot/CTA 중심으로 재배치했다.
- 시장을 장터 배경, 진열대, 대형 거래 오버레이 구조로 다듬었다.
- smoke/visual/capture 검증을 업데이트하고 스크린샷을 저장했다.

## 2. 항구 메인 개선 내용

- 항구 배경을 중심 화면으로 유지하고 어두운 먹색 오버레이 위에 항구명, 설명, CTA를 올렸다.
- `시장 가기`, `출항`, `의뢰`, `어업` 버튼을 hero 내부의 강한 command 버튼으로 배치했다.
- 시설 hotspot 좌표를 재조정했다. 시장/술집은 마을 쪽, 관청은 언덕 쪽, 조선소는 물가/하단 쪽에 더 가깝게 보이도록 했다.
- 오늘의 항구 패널에 특산품, 월간 소식, 추천 행동, 화물 요약을 짧게 표시했다.

## 3. 지도 교체/개선 방식

- `WorldMapBoard`에 `KoreaRouteMapLayer` inline SVG를 추가했다.
- SVG는 서해/남해/동해 라벨, 한반도 실루엣, 산맥/하천 힌트, 제주, 대마도를 포함한다.
- 기존 port `map.x/y` 좌표와 route line 로직은 유지했다.
- 노드를 클릭하면 현재 위치에서 연결된 route를 선택하도록 했다.
- 해로/육로, 현재 위치, 선택 목적지, 추천 판매처, 배 정박지를 기존 시각 언어 위에 더 선명하게 얹었다.

## 4. 시장 화면 개선 내용

- 시장은 항구 배경 위 장터 진열대가 먼저 보이는 구조로 정리했다.
- 상품 아이콘을 크게 보이게 하고, 상품명/가격 라벨은 보조 정보로 줄였다.
- 상품 선택 시 대형 거래 오버레이가 오른쪽에 열리고 살 때 가격, 팔 때 가격, 보유량, 평균 매입가, 현재 손익, 추천 판매처를 보여준다.
- `사기`와 `팔기` 버튼은 큰 command 버튼으로 유지했다.
- 844x390에서 진열대가 얇게 눌리는 문제를 2차 polish에서 수정했다.

## 5. 레퍼런스에서 차용한 구조적 요소

- 장면 우선 화면 구성
- compact top HUD
- 강한 CTA 위계
- 배경 위 overlay panel
- 명령형 action button

원본 이미지, 텍스트, 캐릭터, 세계관, 자산은 사용하지 않았다.

## 6. 조선풍으로 재해석한 요소

- 먹색/남색 반투명 패널, 한지색 장터 패널, 황동 테두리, 청록/금색 CTA를 사용했다.
- 항구/장터/관청/출항/해로/육로 감각을 조선 후기 무역 RPG에 맞게 유지했다.
- 지도는 실제 GIS가 아니라 한반도 전도 감각을 주는 게임용 SVG로 제작했다.

- 상단 튜토리얼 패널과 탭이 아직 화면을 꽤 차지한다. 2차 UI 패스에서 의뢰/튜토리얼/하단 탭을 더 장면형으로 정리할 필요가 있다.
